Mitchell Woite is an aspiring front-end developer born and raised in Buffalo, New York. Coursework

Intro to Web Development

This course taught the basics of web development. Most of the work was done in HTML 4 / CSS 2 as Mitchell created basic websites to learn an understanding of front-end development. This course also gave an introduction on hand-coding websites based on wireframes.

Advanced Web Development

The websites developed in this course were made using HTML 5 / CSS 3. Advanced areas were covered such as making responsive designs, using CSS animations, using frameworks, writing JavaScript, following w3 accesibility standards, and basic uses of PHP, JSON, and APIs.

Digital Strategy and Analytics

This course provided insight into the strategic and analytical side of web development. He was taught how to optimize a website's SEO, navigate Google Analytics, make landing pages, understand PPC advertising, and have effective email marketing strategies.

Capstone / Professional Ethics

For his capstone, Mitchell is working with refugee empowerment organization Syracuse RISE to improve their website and help with their social media. The changes to their website include adding more interactivity and removing dead-links and outdated information. This class also taught Mitchell of the ACM Code of Ethics for IT professionals.
